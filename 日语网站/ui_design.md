# 【Anime日语每日课】用户界面设计

基于《铃芽之旅》海报的清新简洁风格，设计【Anime日语每日课】的用户界面，以支持所有互动学习功能。

## 1. 主界面布局

### 顶部导航栏
- 左侧：系统Logo（使用《铃芽之旅》风格的字体和配色）
- 中间：当前日期和学习进度指示
- 右侧：设置按钮、个人学习数据按钮

### 中央内容区
- 视频播放器（占据主要空间）
- 双语字幕区域（可切换显示/隐藏）
- 今日学习重点提示（如"今日重点：掌握'〜される'的两种用法"）

### 底部控制栏
- 播放/暂停按钮（空格键快捷键）
- 进度条
- 音量控制
- 字幕控制（中文/日文/双语/关闭）
- 播放速度控制
- 全屏按钮

## 2. 互动功能界面

### 点击查词功能
- 鼠标悬停在日语台词上时，显示半透明悬浮卡片
- 卡片内容：假名注音、中文释义、例句
- 卡片样式：圆角矩形，浅蓝色背景，深蓝色文字

### AI慢速复读功能
- 位置：视频播放器右侧
- 样式：圆形按钮，中蓝色背景，白色图标
- 点击后：显示加载动画，生成慢速音频
- 生成完成后：显示播放按钮，点击播放慢速版音频

### 跟读打分功能
- 位置：视频播放器右侧，AI慢速复读按钮下方
- 样式：圆形按钮，中蓝色背景，白色麦克风图标
- 点击后：显示录音倒计时和波形动画
- 录音完成后：显示表情符号评分（😊→优秀 / 🤔→需练习）
- 评分结果：使用动画效果显示，配合简短文字反馈

### 趣味填空功能
- 位置：视频播放完成后自动显示
- 样式：卡片式界面，白色背景，深蓝色边框
- 内容：显示带有空缺的台词，下方提供3-4个选项
- 选项样式：圆角矩形按钮，选中时变色
- 答题反馈：正确答案绿色高亮，错误答案红色标记并显示正确答案

## 3. 知识管理界面

### 碎片复习弹幕
- 样式：半透明文字，从右向左飘过视频画面
- 控制：右上角开关按钮，可随时关闭/开启
- 内容：之前学过的生词和语法点

### 每周学习报告
- 样式：独立页面，清新简洁的卡片式布局
- 内容：
  - 掌握词汇数统计（数字+环形进度图）
  - 学习时长统计（柱状图，每日对比）
  - 进步曲线图（折线图，显示正确率变化）
  - 动漫角色名言（配有角色头像）

## 4. 响应式设计
- 桌面版：完整显示所有功能区域
- 平板版：调整布局，保持主要功能可见
- 移动版：简化界面，关键功能优先显示

## 5. 交互细节
- 所有按钮悬停时有轻微放大效果
- 点击操作有轻微反馈动画
- 学习完成时播放《鬼灭之刃》战斗胜利音效
- 支持键盘快捷键（空格暂停/播放，ESC退出全屏等）
